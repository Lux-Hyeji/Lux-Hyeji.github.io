## [문제](https://www.acmicpc.net/problem/2231)

### 해석
>분해합 = 주어진숫자 + 각자리숫자<br>
>256    = 245    + (2+4+5)<br>
>256의 생성자는 245     

### 소스코드

```python

import sys
n = int(sys.stdin.readline())

for i in range(1,n+1):
    number = list(map(int,str(i)))
    #print(number)
    result = sum(number)
    total = i + result
    if total == n:
        print(i)
        break
    if i == n:
        print(0)

```
